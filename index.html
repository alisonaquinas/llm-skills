Add this marketplace to Claude Code

Use the GitHub repository as the primary marketplace source, then install either of the two published plugins. Individual skills are included within those plugins; they are not listed as standalone plugins.

1. Add marketplace (recommended)/plugin marketplace add alisonaquinas/llm-skills

Marketplace repo: alisonaquinas/llm-skills

2. Install a plugin
Shared Skills
/plugin install shared-skills@llm-skills
CI/CD Skills
/plugin install ci-cd@llm-skills
Marketplace URL (secondary option)/plugin marketplace add https://alisonaquinas.github.io/llm-skills/marketplace.json

Claude Plugin Marketplace

This marketplace publishes two installable Claude Code plugins, exposes their bundled skills as static searchable pages, and ships machine-readable artifacts like marketplace.json, rss.xml, and sitemap.xml for discovery.

Shared Skillsv1.5.2

shared-skills

Cross-compatible LLM skills for Claude Code and Codex: ag, aws, az, bash, changelog, claude-docs, claude-cli-docs, chatgpt-docs, codex-docs, codex-cli-docs, docker, git, glab, markdownlint, powershell, sqlite, zoxide, and skill-creator.

Install
/plugin install shared-skills@llm-skills
