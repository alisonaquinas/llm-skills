Alison's LLM Plugins

This marketplace publishes installable Claude Code plugins, exposes their bundled skills as static searchable pages, and ships machine-readable artifacts like marketplace.json, rss.xml, and sitemap.xml for discovery.

Add this marketplace to Claude Code

Use the GitHub repository as the primary marketplace source, then install one of the published plugins. Individual skills are included within those plugins; they are not listed as standalone plugins.

1. Add marketplace (recommended)
/plugin marketplace add alisonaquinas/llm-skills

Marketplace repo: alisonaquinas/llm-skills

2. Install a plugin
Shared Skills
/plugin install shared-skills@llm-skills
CI/CD Skills
/plugin install ci-cd@llm-skills
Software Design Skills
/plugin install software-design@llm-skills
Doc Skills
/plugin install doc-skills@llm-skills
Marketplace URL (secondary option)
/plugin marketplace add https://alisonaquinas.github.io/llm-skills/marketplace.json
