Doc Skills for Claude Code and Codex

Cross-compatible LLM skills for document workflows, text-first publishing, host-specific Markdown, and diagram authoring: OOXML, PDF, Pandoc, LaTeX, Typst, Markdown, AsciiDoc, Mermaid, PlantUML, and Graphviz.

Install the Doc Skills bundle

Add the marketplace once, then install just this bundle. A bundle ships all 22 skills listed below.

Claude Code
/plugin install doc-skills@llm-skills
Codex
codex plugin marketplace add alisonaquinas/llm-skills

Then open /plugins and install doc-skills.
