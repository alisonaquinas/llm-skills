CI/CD Skills for Claude Code and Codex

Cross-compatible LLM skills for CI/CD pipelines: GitHub Actions, GitLab CI, Jenkins, Docker, Kubernetes, and deployment workflows.

Install the CI/CD Skills bundle

Add the marketplace once, then install just this bundle. A bundle ships all 61 skills listed below.

Claude Code
/plugin install ci-cd@llm-skills
Codex
codex plugin marketplace add alisonaquinas/llm-skills

Then open /plugins and install ci-cd.
