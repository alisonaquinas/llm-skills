Web Design Skills for Claude Code and Codex

Cross-compatible LLM skills for modern web design, frontend frameworks, backend frameworks, static sites, UI systems, and web build tooling.

Install the Web Design Skills bundle

Add the marketplace once, then install just this bundle. A bundle is a single Claude Code plugin that ships all 16 skills listed below.

Install command
/plugin install web-design-skills@llm-skills
